Toggle Dark Mode

This was my first ever true coding project, at this point i had learnt around 2 weeks of javascript.

For a first timer this project, all though looks simple at the base of it, was actually quite tricky. My plan was simple, create a function so that when a icon is clicked, change the whole screen to black and vice versa, essentially a toggle dark mode.

I first started off by creating the neccesary functions to change the screen to black, which if you have looked at the code, looks simple but took me quite some time to figure out. Although i didnt plan for it, i also then decided to add a real-time clock by using the date method in Javascript.

This project was incredibly satisfiying to finish as it was my first one and it came together quite nicely.
